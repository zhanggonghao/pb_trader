## 模块1：周期净值信息

### 1.1 净值增长率

**公式：**
```
净值增长率 = (期末净值 - 期初净值) / 期初净值
```

**说明：**
- 期初净值：报告周期前一交易日的净值
- 期末净值：报告周期最后一个交易日的净值

### 1.2 基准收益率

**公式：**
```
基准收益率 = (期末基准价格 - 期初基准价格) / 期初基准价格
```

### 1.3 最大回撤

**公式：**
```
最大回撤 = max( (净值峰值 - 当前净值) / 净值峰值 )
```

**说明：**
- 净值峰值：从报告开始到当前日期的最大净值
- 计算周期内所有日期的回撤，取最大值

## 模块2：每日收益明细

### 2.1 每日收益率

**公式：**
```
每日收益率 = (当日净值 - 前一日净值) / 前一日净值
```

### 2.2 基准日收益率

**公式：**
```
基准日收益率 = (当日基准价格 - 前一日基准价格) / 前一日基准价格
```

## 模块3：多空详情

### 3.1 股票组合收益

**公式：**
```
股票组合收益 = Σ(各证券账户收益)
```

### 3.2 期货收益

**数据来源：** 直接从期货数据文件读取

### 3.3 基差收益率

**公式：**
```
基差收益率 = 期货收益率 + 基准收益率
```

**说明：**
- 期货收益率 = 期货收益 / 期货名义市值

### 3.4 期货名义市值

**公式：**
```
期货名义市值 = 期货保证金 / 保证金比例
```

**参数：**
- 保证金比例：配置参数 `FUTURE_MARGIN_RATIO`，默认0.12

### 3.5 股票贡献

**公式：**
```
股票贡献 = 股票组合收益 / 产品总净资产
```

### 3.6 期货基差贡献

**公式：**
```
期货基差贡献 = (期货收益 + 期货名义市值 × 基准收益率) / 产品总净资产
```

**说明：**
此公式等价于：`(期货收益 - 期货名义市值 × 基准收益率 × -1) / 产品总净资产`

### 3.7 总贡献

**公式：**
```
总贡献 = 股票贡献 + 期货基差贡献
```

## 模块4：每日个股权重Top10

### 4.1 数据合并

**处理步骤：**
1. 合并同一股票在不同证券账户的持仓
   ```
   合并持仓 = Σ(各账户同一股票持仓数量)
   ```

### 4.2 股票市值计算

**公式：**
```
股票市值 = 持仓数量 × 股票价格
```

**数据来源：**
- 持仓数量：从持仓文件读取
- 股票价格：从米筐获取（前复权）

### 4.3 权重计算

**公式：**
```
股票权重 = 股票市值 / 股票总市值
```

### 4.4 Top10筛选

**处理步骤：**
1. 按权重降序排序
2. 取前10名

## 模块5：持仓集中度

### 5.1 集中度计算

**公式：**
```
前N%股票累计权重 = Σ(权重排序前N%股票的权重)
```

**参数：**
- N：配置参数 `CONCENTRATION_TOP_PCT`，默认20%

**说明：**
- N%按股票数量计算，例如280只股票时，前20%即前56只股票
- 计算这些股票的累计权重占比

## 模块6：因子风格敞口

### 6.1 组合因子暴露

**公式：**
```
组合因子暴露 = Σ(股票因子暴露 × 股票权重)
```

**说明：**
- 股票因子暴露：从米筐获取的单个股票在该因子上的暴露值
- 股票权重：该股票市值占组合总市值的比例

### 6.2 基准因子暴露

**数据来源：** 从米筐获取沪深300指数的因子暴露

### 6.3 因子列表

| 因子代码 | 中文名称 | 说明 |
|---------|---------|------|
| size | 规模 | 股票市值大小 |
| non_linear_size | 非线性规模 | 市值的非线性变换 |
| momentum | 动量 | 价格趋势 |
| liquidity | 流动性 | 交易活跃度 |
| book_to_price | 价值 | 市净率相关 |
| leverage | 杠杆 | 财务杠杆水平 |

| earnings_yield | 盈利 | 盈利能力 |

## 股票代码转换规则

### 规则说明

| 股票代码前缀 | 市场 | 后缀 |
|------------|------|------|
| 60 | 沪市 | XSHG |
| 688 | 科创板 | XSHG |
| 其他 | 深市 | XSHE |

### 转换逻辑

```python
def convert_stock_code(code):
    code_padded = str(code).zfill(6)
    if code_padded.startswith('60') or code_padded.startswith('688'):
        return f"{code_padded}.XSHG"
    else:
        return f"{code_padded}.XSHE"
```
